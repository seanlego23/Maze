# Maze

This is a 3D Maze generator using PCG and PCGEx Unreal Engine plugins.

## Prerequisites

- UE 5.7
- PCG Plugin enabled
- [PCGEx Plugin](https://github.com/PCGEx/PCGExtendedToolkit)

## Wanted Features

- Stairs (Working on)
- Elevators
	- One-Way
	- Two-Way
- Intermittent Paths
- Open Areas/Volumes
